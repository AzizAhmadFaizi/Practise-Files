Git Note:

1. Get Version INFO.
   git --version

2. Status INFO
   git status

3. Make this repository as git
   git init

4. All directions add to stage level
   git add .

5. ADD to Main 
   git commit -m "Version 1"

6. To add the username 
git config --global user.name "Aziz Ahmad"

7. Add the email to git
  git config --global user.email "azizahmad.faizi10@gmail.com"

8. Get information about saving version
  git log

9. Send the File to origin
  git remote add origin (SSD address)

10. Push the content of files
  git push -u origin master
